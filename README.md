# SDK-Generator

A powerful and modern SDK generator built for Unreal Engine projects.

## Overview

SDK-Generator parses Unreal Engine game data and produces a clean, accessible SDK structure for development, tooling, analysis, and reverse-engineering research.

## Features

* Automatic SDK generation from Unreal Engine assets
* Fast, optimized parsing
* Clean and organized output structure
* Ideal for learning, analysis, and tooling development

## How It Works

1. Loads SDK-Generator to analyze the engine's internal data.
2. The tool gathers structures, classes, functions, and relevant engine metadata.
3. Output is formatted into a readable, ready-to-use SDK folder.

## Notes

This project is designed strictly for educational use, engine research, and tool development. Use it responsibly within legal and ethical boundaries.
